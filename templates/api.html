UISM.link API

How to use API of UISM.link

To call UISM.link just use the following URL:
https://uism.link/?link=YOUR_LONG_LINK
Replace YOUR_LONG_LINK with the link you want to shorten.

Example

curl "https://uism.link/?link=https://example.com/"

Response will be like this:

http://uism.link/dww3el
